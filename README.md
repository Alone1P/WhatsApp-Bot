# بوت واتساب المتقدم للتحكم في المجموعات والملف الشخصي

بوت واتساب شامل ومتقدم يوفر إدارة كاملة للمجموعات مع العديد من الوظائف المتطورة.

## المميزات الرئيسية

### 🔧 أوامر الملف الشخصي:
- تغيير اسم البوت
- تغيير صورة الملف الشخصي للبوت

### 👥 إدارة المجموعات:
- تغيير اسم وصورة ووصف المجموعة
- رفع وتنزيل المشرفين (بالرد أو ذكر الرقم)
- طرد الأعضاء (بالرد أو ذكر الرقم)
- منشن جماعي لجميع الأعضاء
- تثبيت وإلغاء تثبيت الرسائل

### 📊 إدارة النشاط:
- تصفية الأعضاء غير النشطين (لم يرسلوا رسائل منذ يومين)
- عرض الأصنام (غير النشطين منذ يوم)
- إحصائيات شاملة للمجموعة

### ⚠️ نظام التحذيرات والأمان:
- نظام تحذيرات (3 تحذيرات = طرد تلقائي)
- كتم مؤقت للأعضاء
- مكافحة السبام
- فلترة الروابط

### 🎯 الوظائف التفاعلية:
- رسائل ترحيب تلقائية للأعضاء الجدد
- حفظ وعرض قواعد المجموعة
- جدولة الرسائل لأوقات محددة
- نظام التذكيرات

### 📊 الاستطلاعات:
- إنشاء استطلاعات رأي
- نظام تصويت متقدم
- عرض النتائج بالنسب المئوية

### 🎮 الترفيه والخدمات:
- معلومات الطقس للمدن
- ترجمة فورية للرسائل
- ألعاب بسيطة (حجر ورقة مقص)

## التثبيت والتشغيل

### المتطلبات:
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn
- حساب واتساب نشط

### خطوات التثبيت:

1. **استنساخ المستودع:**
```bash
git clone <repository-url>
cd whatsapp_bot
```

2. **تثبيت التبعيات:**
```bash
npm install
```

3. **تشغيل البوت:**
```bash
npm start
```

4. **مسح رمز QR:**
   - عند تشغيل البوت لأول مرة، سيظهر رمز QR في الطرفية
   - افتح واتساب على هاتفك
   - اذهب إلى الإعدادات > الأجهزة المرتبطة
   - امسح رمز QR المعروض

## دليل الأوامر الشامل

جميع الأوامر تبدأ بعلامة التعجب (!)

### 🔧 أوامر الملف الشخصي:
```
!changename [الاسم الجديد] - تغيير اسم البوت
!changeprofilepic - تغيير صورة البوت (أرسل صورة مع الأمر)
```

### 🏠 أوامر المجموعات الأساسية:
```
!changegroupname [الاسم الجديد] - تغيير اسم المجموعة
!changegrouppic - تغيير صورة المجموعة (أرسل صورة مع الأمر)
!changegroupdesc [الوصف الجديد] - تغيير وصف المجموعة
```

### 👥 إدارة الأعضاء:
```
!رفع / !promote - ترقية عضو لمشرف (بالرد على رسالته أو ذكر رقمه)
!تنزيل / !demote - تنزيل مشرف لعضو عادي
!طرد / !kick - طرد عضو من المجموعة
!منشن / !mentionall - منشن جماعي لجميع الأعضاء
!تصفية / !cleanup - طرد الأعضاء غير النشطين (لم يرسلوا رسائل منذ يومين)
!اصنام / !inactive - عرض الأعضاء غير النشطين (منذ يوم واحد)
```

### 📌 إدارة الرسائل:
```
!تثبيت / !pin - تثبيت رسالة (بالرد عليها)
!الغاء_تثبيت / !unpin - إلغاء تثبيت رسالة
```

### 📊 الإحصائيات والمعلومات:
```
!احصائيات / !stats - عرض إحصائيات شاملة للمجموعة
!ترحيب [رسالة الترحيب] - تعيين رسالة ترحيب للأعضاء الجدد
!قواعد [نص القواعد] - حفظ قواعد المجموعة
!عرض_قواعد / !showrules - عرض قواعد المجموعة
```

### ⚠️ نظام التحذيرات والأمان:
```
!تحذير / !warn - إعطاء تحذير لعضو (بالرد على رسالته)
!كتم [عدد الدقائق] / !mute - كتم عضو مؤقتاً
!مكافحة_سبام / !antispam - تفعيل/إلغاء مكافحة السبام
!فلترة_روابط / !linkfilter - تفعيل/إلغاء فلترة الروابط
```

### ⏰ الجدولة والتذكيرات:
```
!جدولة [HH:MM] [نص الرسالة] - جدولة رسالة لوقت محدد
!تذكير [HH:MM] [نص التذكير] - تعيين تذكير لوقت محدد
```

### 📊 الاستطلاعات:
```
!استطلاع [نص السؤال] / !poll - إنشاء استطلاع رأي
!نعم [ID] / !yes - التصويت بنعم
!لا [ID] / !no - التصويت بلا
!نتائج [ID] / !results - عرض نتائج الاستطلاع
```

### 🎮 الترفيه والخدمات:
```
!طقس [اسم المدينة] / !weather - معلومات الطقس
!ترجم [اللغة المستهدفة] / !translate - ترجمة رسالة (بالرد عليها)
!حجر / !rock - لعبة حجر ورقة مقص
```

### ℹ️ أوامر عامة:
```
!help / !مساعدة - عرض قائمة الأوامر
!status - عرض حالة البوت والإحصائيات
```

## API Endpoints

البوت يوفر أيضاً API endpoints للمراقبة:

- `GET /status` - حالة البوت ومعلومات النشاط
- `GET /qr` - الحصول على رمز QR للمصادقة

## الاستضافة والنشر

### الاستضافة المحلية:
```bash
npm start
```

### الاستضافة على Heroku:
1. إنشاء تطبيق Heroku جديد
2. ربط المستودع بـ Heroku
3. إضافة buildpack لـ Node.js
4. تعيين متغير البيئة `PORT`
5. نشر التطبيق

### الاستضافة على Railway:
1. ربط المستودع بـ Railway
2. إعداد متغيرات البيئة
3. نشر التطبيق تلقائياً

### الاستضافة على VPS:
```bash
# تثبيت PM2 لإدارة العمليات
npm install -g pm2

# تشغيل البوت في الخلفية
pm2 start bot.js --name whatsapp-bot

# حفظ التكوين
pm2 save
pm2 startup
```

## الميزات المتقدمة

### نظام التحذيرات:
- تحذير تلقائي للأعضاء المخالفين
- طرد تلقائي بعد 3 تحذيرات
- تتبع تحذيرات كل عضو

### مكافحة السبام:
- منع الرسائل المتكررة
- حذف الرسائل المشبوهة تلقائياً

### فلترة الروابط:
- منع نشر الروابط في المجموعة
- حذف الروابط تلقائياً

### الجدولة الذكية:
- جدولة رسائل لأوقات محددة
- نظام تذكيرات متقدم
- تنفيذ تلقائي في الوقت المحدد

## ملاحظات مهمة

### الصلاحيات:
- البوت يحتاج إلى صلاحيات المشرف في المجموعات للتحكم الكامل
- بعض الوظائف تتطلب أن يكون البوت مشرفاً

### الأمان:
- لا تشارك ملفات الجلسة (`.wwebjs_auth`) مع أي شخص
- استخدم البوت بحذر في المجموعات المهمة
- تأكد من أن لديك صلاحيات كافية قبل استخدام أوامر الإدارة

### الأداء:
- البوت يحفظ البيانات في الذاكرة (يُفقد عند إعادة التشغيل)
- لحفظ دائم، يمكن إضافة قاعدة بيانات خارجية

## استكشاف الأخطاء

### مشاكل شائعة:
1. **رمز QR لا يظهر**: تأكد من تشغيل البوت في بيئة تدعم عرض النص
2. **البوت لا يرد**: تحقق من الاتصال بالإنترنت وحالة واتساب ويب
3. **أوامر لا تعمل**: تأكد من أن البوت مشرف في المجموعة

### سجلات الأخطاء:
```bash
# عرض سجلات PM2
pm2 logs whatsapp-bot

# عرض حالة العمليات
pm2 status
```
